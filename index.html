Road Bomber

Road Bomber is a fast-paced endless runner where the player must avoid enemies and environmental hazards while traveling as far as possible. The game features dynamic obstacle spawning, scrolling terrain, and collectible coins that increase the player’s score. The project was built using HTML5 Canvas and JavaScript, with an object-oriented architecture separating entities such as the player, enemies, platforms, and collectibles. Collision detection, procedural obstacle generation, and responsive canvas resizing were implemented to create smooth gameplay across different screen sizes. This project demonstrates my experience with game loops, collision systems, and modular JavaScript design.
